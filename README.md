AI设计工具
## 项目简介
这是一个基于AI的设计辅助工具，用于帮助设计学习者快速生成设计思路、配色方案、版式建议和视觉关键词。

## 功能规划
- 输入设计主题（如：咖啡品牌 / 科技产品 / 海报设计）
- AI生成设计方向建议
- 自动输出配色方案
- 自动生成版式结构建议
- 提供AI绘图提示词（用于Midjourney / SD等工具）

## 目标用户
- 视觉传达设计学生
- UI/UX设计初学者
- 需要快速获取设计灵感的人

## 项目目标
通过AI辅助设计流程，提高设计效率，降低灵感获取成本，并作为个人作品集项目展示。
